# Ktor Mastery Course: Habit Tracker API

Welcome to the Ktor Mastery Course! This repository contains a running "Habit Tracker" REST API built with Kotlin and Ktor.

It is designed to take you from zero backend knowledge to full mastery of Ktor.

## Modules Completed So Far:
*   **Module 00:** Environment Setup (Gradle, Netty engine)
*   **Module 01:** Ktor & Coroutines Fundamentals
*   **Module 02:** Routing (Paths, Parameters)
*   **Module 03:** Requests, Responses, and JSON Serialization
*   **Module 04:** Building a Real REST API (CRUD & StatusPages)

## How to run
1. Open this project in IntelliJ IDEA or your preferred IDE.
2. Run `./gradlew run` from the terminal.
3. The API will be available at `http://localhost:8080`.
