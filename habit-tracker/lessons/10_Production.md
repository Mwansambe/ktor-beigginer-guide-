# Module 10: The Polish (Production Concerns)

## CORS & Rate Limiting
*   **CORS:** Cross-Origin Resource Sharing. Tells the browser which websites are allowed to call our API.
*   **Rate Limiting:** Prevents a single user from crashing our server by making 10,000 requests a second.
*   **Swagger:** We added the `swaggerUI` plugin, which reads our `documentation.yaml` and builds a beautiful webpage for testing our API visually!
